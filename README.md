# Аннотация
Страница посвящена списку библиотек разработчика-энтузиаста для убодной работы с компонентами на платах Arduino. Для удобства все библиотеки обозначены в ссылках. Найти интересующую библиотеку можно через Ветки (Branches) репозитория.

# Загрузка библиотеки
1. Скачать .zip файл со страницы с библиотекой
2. В среде Arduino IDE перейти во вкладку _Sketch_ > _Include Library_ > _Add .ZIP Library..._
3. Выбрать в проводнике скачанный .zip файл библиотеки
4. Подтвердить открытие

# Подключение библиотеки в Arduino IDE
1. Перейти во вкладку _Sketch_ > _Include Library_
2. В выпавшем списке найти раздел _Contributed libraries_
3. Выбрать необходимую библиотеку по названию
